# Liferay Connectors

## Outbound

Connector to create or modify custom objects, and any other entities on Liferay.

This Connector reuses the base implementation of
[HTTP JSON Connector](https://github.com/camunda/connectors/blob/main/connectors/http/rest/README.md)
by providing a compatible element template.

## Inbound

Connector to start process instances on Liferay events. This connector is based
on inbound webhooks.
