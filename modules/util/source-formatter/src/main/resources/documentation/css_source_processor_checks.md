# Checks for .css or .scss

Check | Category | Description
----- | -------- | -----------
CSSCommentsCheck | [Styling](styling_checks.md#styling-checks) | Validates comments in `.css` files. |
CSSImportsCheck | [Styling](styling_checks.md#styling-checks) | Sorts and groups imports in `.css` files. |
CSSPropertiesOrderCheck | [Styling](styling_checks.md#styling-checks) | Sorts properties in `.css` files. |
WhitespaceCheck | [Styling](styling_checks.md#styling-checks) | Finds missing and unnecessary whitespace. |
