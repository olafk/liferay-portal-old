# Checks for .markdown or .md

Check | Category | Description
----- | -------- | -----------
MarkdownBreakingChangesAmendmentsFileCheck | [Styling](styling_checks.md#styling-checks) | Checks that breaking changes amendments should contain the schematized breaking changes in `BREAKING_CHANGES_AMENDMENTS.md` file. |
MarkdownEmptyLinesCheck | [Styling](styling_checks.md#styling-checks) | Finds missing and unnecessary empty lines. |
MarkdownFileExtensionCheck | [Styling](styling_checks.md#styling-checks) | Finds `markdown` files with `.markdown` extension (use `.md`). |
MarkdownSourceFormatterDocumentationCheck | [Documentation](documentation_checks.md#documentation-checks) | Validates the header of Source Formatter documentation files. |
MarkdownSourceFormatterReadmeCheck | [Documentation](documentation_checks.md#documentation-checks) | Generates Source Formatter documentation index files. |
MarkdownStylingCheck | [Styling](styling_checks.md#styling-checks) | Applies rules to enforce consistency in code style. |
MarkdownWhitespaceCheck | [Styling](styling_checks.md#styling-checks) | Finds missing and unnecessary whitespace. |
